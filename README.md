# Easy-cad
AI system that generates architectural AutoCAD floor plans from user inputs such as land size, building codes, floors, rooms, and parking needs. It optimizes space use, ensures compliance with regulations, and produces fast design alternatives. The tool reduces manual drafting time and supports export to CAD formats for architects and developers.
AI Architectural AutoCAD Generator
Final project for the Building AI course
Summary
This project is an AI system that automatically generates architectural AutoCAD floor plans based on user inputs such as land size, building codes, number of floors, rooms, and parking requirements. It produces optimized and regulation-compliant design alternatives quickly and efficiently.
Background
This project addresses the challenge of time-consuming and complex architectural drafting. Manual floor plan design requires significant effort, expertise, and compliance with building regulations.
Problems it solves:
Slow manual drafting of architectural plans
Difficulty in optimizing space efficiently
Risk of human error in building code compliance
High cost of early-stage design in architecture
This idea is important because it can significantly speed up the architectural design process and support engineers and developers in making faster, smarter decisions.
My motivation is to simplify architectural design using AI and make early-stage planning more accessible and efficient.
How is it used?
The user enters project requirements such as land dimensions, number of floors, rooms, parking needs, and building regulations. The AI system processes the input and generates one or more optimized architectural floor plans.
This tool is used in:
Early design stages of construction projects
Architectural offices
Real estate development planning
Users include architects, engineers, and real estate developers.
The output can be exported as CAD-compatible files for further editing in AutoCAD.
Data sources and AI methods
The system uses synthetic and real architectural design rules, including building codes and spatial optimization principles.
Possible AI methods:
Machine learning regression models
Rule-based systems for building codes
Optimization algorithms for space planning
Data sources may include:

Challenges
The project does not fully replace professional architects and may not handle complex or highly customized architectural designs perfectly.
Limitations include:
Dependence on accurate input data
Complexity of real-world building regulations
Ethical concerns about replacing human design judgment
